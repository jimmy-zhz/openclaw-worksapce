# Research Directory

## Purpose
This directory contains research materials, analysis, and findings related to the MPNP immigration project. Organized research supports evidence-based decision making throughout the immigration process.

## Research Methodology

### 1. Source Evaluation
- **Primary Sources:** Government websites (IRCC, Manitoba Immigration)
- **Secondary Sources:** Immigration consultants, legal experts
- **Tertiary Sources:** News articles, forums, community insights
- **Academic Sources:** Research papers, statistical reports

### 2. Data Collection
- **Systematic:** Regular monitoring of key sources
- **Comprehensive:** Multiple perspectives on each topic
- **Documented:** Source citations and collection dates
- **Organized:** Categorized by topic and relevance

### 3. Analysis Framework
- **Comparative Analysis:** Different options evaluated against criteria
- **Risk Assessment:** Identified risks and mitigation strategies
- **Cost-Benefit Analysis:** Financial and non-financial factors
- **Timeline Impact:** How research affects project timeline

## Documentation Standards

### File Naming:
```
YYYY-MM-DD_Topic_Summary_Source.ext
```

Examples:
- `2026-04-21_MPNP-Points-Calculation_Current-Criteria_IRCC.pdf`
- `2026-04-21_AI-Job-Market_Winnipeg-2026_StatisticsCanada.csv`
- `2026-04-21_PGWP-Eligibility_Diploma-Programs_UniversityWebsite.md`

### Citation Format:
For each research finding, include:
1. Source URL or reference
2. Date accessed
3. Author/organization
4. Key findings summary
5. Relevance to project

## Quality Assurance

### Research Validation:
1. **Cross-reference:** Multiple sources for critical information
2. **Expert Consultation:** Professional advice for complex matters
3. **Government Verification:** Official sources for policy information
4. **Community Feedback:** Real-world experiences from others

### Bias Awareness:
- Recognize potential biases in sources
- Consider multiple perspectives
- Distinguish between fact and opinion
- Note uncertainties and gaps in information
